The Labs

  • Week 2 Lab

    In the second week of the module we recieved our first introduction into the software visual studio code.
    We began by making a simple index.html within which we added some simple element tags like <ul>, <img>, and also we added different font sizes with the <h1>, <h2> tags.

  • Week 4 Lab

    By week 4 we have been introduced to the concept of semantics for our HTML code structures. We learned some new tags for HTML these involved <article>, <nav>, <section>, <header> and many more.
    Most Importantly we began using CSS. By linking it to our HTML files we were able to add designs through the use of recalling the semantic tags from HTML within the CSS.

  • Week 7 Lab

    Up until this point we have been limited in our web pages flexibilities in terms of different resolutions and aspect ratios of monitors.
    This changed with this lab in which we have been assigned to utilise the flex functions in CSS.
    These functions made styling far less tedious and helped in compensating for different screen sizes.

  • Week 8 Lab

    The final practical lab of the module applied more forms of the flex functions in CSS.
    It was almost purely graded on our CSS apart from one portion being HTML which involved the use of mapping.
    The mapping applied a range of pixels which in combination with an <a> tag on an <img> tag created a linked image.
